## 题目1：朋友圈

**描述**
班上有 N 个学生，有些学生之间是朋友关系。朋友关系可以传递，如果 A 认识 B，B 认识 C，那么 A 也间接认识 C。我们想划分朋友圈：即最大的群体，使得群体内任何两个学生都直接或间接是朋友。给定一个 N × N 的矩阵 M 表示学生之间的朋友关系，如果 `M[i][j] = 1`，表示第 i 个和第 j 个学生互为朋友，否则为 0。注意：自己与自己也是朋友（`M[i][i] = 1`）。请你计算班上有多少个不同的朋友圈（即连通分量数）。

**输入**

- 第一行一个整数 N（1 ≤ N ≤ 200）
- 接下来 N 行，每行 N 个整数，表示矩阵 M。矩阵是对称的，且对角线为 1。

**输出**
一个整数，表示朋友圈的数量。

**样例**

```text
输入：
3
1 1 0
1 1 0
0 0 1
输出：
2
```

**解释**：学生0和学生1是朋友，形成一个朋友圈；学生2自己形成一个朋友圈。

**提示**
使用并查集，初始每个学生各自为一个集合，然后遍历矩阵的上三角，若 `M[i][j] = 1`，则合并 i 和 j。最后统计有多少个不同的根。

## 题目2：冗余连接

**描述**
在本问题中，树指的是一个连通且无环的无向图。输入一个图，该图由一个有着 N 个节点（节点值 1~N）的树添加一条额外的边构成（因此共有 N 条边）。添加的边有两个顶点，且不会自环。请找到一条可以删除的边，删除后使剩余部分成为一棵树。如果有多个答案，则返回输入中最后出现的那条边。

**输入**

- 第一行一个整数 N（2 ≤ N ≤ 1000）
- 接下来 N 行，每行两个整数 u, v，表示一条边。

**输出**
两个整数，即要删除的边（u v，空格隔开），按输入时的顺序输出。

**样例**

```text
输入：
4
1 2
2 3
3 4
1 4
输出：
1 4
```

**解释**：删除 1-4 后，1-2-3-4 成为一棵树。或者删除 1-2 也可以，但 1-4 是最后出现的。

**提示**
使用并查集，依次处理每条边，如果两个顶点已经在同一个集合中，说明这条边是冗余的，记录为答案。最后一条满足条件的边即为所求。

## 题目3：等式方程的可满足性

**描述**
给定一个由字符串组成的数组 equations，每个字符串表示一个变量之间的关系，长度为 4，例如 `"a==b"` 或 `"a!=b"`。每个变量是小写字母（a-z）。判断是否可以为所有变量赋予整数（或布尔值），使得所有给定的关系同时成立。如果可能，返回 true；否则返回 false。

**输入**

- 第一行一个整数 n（1 ≤ n ≤ 500）
- 接下来 n 行，每行一个等式方程，如 `"a==b"` 或 `"a!=b"`。

**输出**
`true` 或 `false`。

**样例1**

```text
输入：
2
a==b
b!=a
输出：
false
```

**解释**：a 必须等于 b，但又要求 b 不等于 a，矛盾。

**样例2**

```text
输入：
3
a==b
b==c
c!=a
输出：
false
```

**解释**：a==b, b==c ⇒ a==c，但 c!=a 矛盾。

**样例3**

```text
输入：
3
a==b
b==c
c==a
输出：
true
```

**提示**
首先将所有相等关系（`==`）的变量合并到同一个集合中。然后检查所有不等关系（`!=`）：如果两个变量在同一个集合中，则矛盾，返回 false；否则最终返回 true。

## 题目4：岛屿数量 II（进阶）

**描述**
给定一个二维网格，初始时全是水域（0）。然后按照顺序每次将一个位置 (row, col) 变成陆地（1）。每次操作后，你需要返回当前网格中岛屿的数量。岛屿定义为由相邻（上下左右）陆地形成的连通分量。注意：网格边界外的位置视为水域。

**输入**

- 第一行三个整数 m, n, k，分别表示网格的行数、列数，以及操作的次数。
- 接下来 k 行，每行两个整数 r, c，表示将 (r, c) 变成陆地（保证之前是水域）。
  m, n 最大为 10^4，k 最大为 10^4。

**输出**
k 行，每行一个整数，表示每次操作后的岛屿数量。

**样例**

```text
输入：
3 3 4
0 0
0 1
1 2
2 1
输出：
1
1
2
3
```

**解释**：

- 第一次添加 (0,0)，岛屿1
- 第二次添加 (0,1)，与 (0,0) 相邻，岛屿仍为1
- 第三次添加 (1,2)，单独一个，岛屿2
- 第四次添加 (2,1)，单独一个，但注意是否与前面相邻？(2,1) 与 (1,2) 对角不相邻，所以岛屿3。

**提示**
动态添加陆地，使用并查集维护连通分量。每次添加时，将新位置视为新岛屿，然后检查四个方向，如果相邻位置是陆地且不在同一集合，则合并并减少岛屿计数。

## 题目5：带权并查集 - 食物链

**描述**
动物王国中有三类动物 A, B, C，这三类动物的食物链构成环形：A 吃 B，B 吃 C，C 吃 A。现有 N 个动物，编号 1~N。每个动物都是 A, B, C 中的一种，但我们并不知道具体是哪一种。有人用两种说法对这 N 个动物构成的食物链关系进行描述：

- 第一种说法是 "1 X Y"，表示 X 和 Y 是同类。
- 第二种说法是 "2 X Y"，表示 X 吃 Y。
  此人对 N 个动物，用上述两种说法，一句接一句地说出 K 句话，这 K 句话有的是真的，有的是假的。当一句话满足下列三条之一时，这句话就是假话，否则是真话：

1. 当前的话与前面的某些真话冲突，是假话；
2. 当前的话中 X 或 Y 比 N 大，就是假话；
3. 当前的话表示 X 吃 X，就是假话。
   你的任务是根据给定的 N 和 K 句话，输出假话的总数。

**输入**

- 第一行是两个整数 N（1 ≤ N ≤ 50000）和 K（0 ≤ K ≤ 100000）。
- 接下来 K 行，每行三个整数 D, X, Y，描述一句话，其中 D 表示说法种类。

**输出**
一个整数，表示假话的总数。

**样例**

```text
输入：
100 7
1 101 1
2 1 2
2 2 3
2 3 3
1 1 3
2 3 1
1 5 5
输出：
3
```

**解释**：假话是第1句（X超出范围）、第4句（X吃X）、第5句（与前面冲突）等，最终总数为3。

**提示**
经典带权并查集问题，需要维护每个节点与根节点的关系（同类、吃、被吃）。可以使用并查集，每个节点维护到父节点的权值（0：同类，1：吃父节点，2：被父节点吃），路径压缩时更新权值。在合并时根据关系推导权值。
