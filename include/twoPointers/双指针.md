## 题目一：最接近目标值的两数之和

### 问题描述

给定一个升序排列的整数数组 `a` 和一个目标整数 `target`，请你在数组中找出两个数，使得它们的和与 `target` 最接近。输出这个最接近的和。

如果有多个不同的数对满足条件，输出任意一个合法结果即可。

### 输入格式

第一行包含两个整数 `n` 和 `target`，分别表示数组长度和目标值。
第二行包含 `n` 个整数，表示升序数组 `a`。

### 输出格式

一个整数，表示与 `target` 最接近的两数之和。

### 样例输入

```text
5 10
1 2 4 7 9
```

### 样例输出

```text
11
```

### 样例解释

数组中的数对 (2,9) 的和为 11，与 10 相差 1；数对 (1,9) 的和为 10，与 10 相差 0。因此最接近的和是 10（若存在和等于 target 则输出 target）。

### 数据范围

- $2 \leq n \leq 10^5$
- $-10^9 \leq a[i], target \leq 10^9$

### 提示

使用左右指针，根据当前和与 target 的大小关系移动指针，同时记录最接近的和。

## 题目二：最长不含重复字符的子串

### 问题描述

给定一个字符串 `s`，请你找出其中不含有重复字符的 **最长子串** 的长度。

### 输入格式

一行字符串 `s`，仅由小写英文字母组成。

### 输出格式

一个整数，表示最长无重复字符子串的长度。

### 样例输入

```text
abcabcbb
```

### 样例输出

```text
3
```

### 样例解释

无重复字符的最长子串是 `"abc"`，长度为 3。

### 数据范围

- $1 \leq |s| \leq 10^5$

### 提示

维护滑动窗口，使用双指针 `l` 和 `r`，并用一个数组或哈希表记录窗口内字符出现次数。当新字符重复时移动左指针直到重复消除。

## 题目三：三数之和为零

### 问题描述

给定一个包含 `n` 个整数的数组 `nums`，判断是否存在三个元素 `a, b, c` 使得 `a + b + c = 0`？请找出所有 **不重复** 的三元组。

注意：答案中不可以包含重复的三元组。

### 输入格式

第一行一个整数 `n`，表示数组长度。
第二行 `n` 个整数，表示数组 `nums`。

### 输出格式

输出若干行，每行三个整数，表示一个和为 0 的三元组，按升序排列（即三元组内 `a ≤ b ≤ c`），且所有三元组按第一个数升序、第一个数相同时按第二个数升序输出。
如果没有满足条件的三元组，输出 `"No solution"`。

### 样例输入

```text
6
-1 0 1 2 -1 -4
```

### 样例输出

```text
-1 -1 2
-1 0 1
```

### 数据范围

- $3 \leq n \leq 3000$
- $-10^5 \leq nums[i] \leq 10^5$

### 提示

先对数组排序，然后固定第一个数，用双指针在剩余区间内寻找两数之和等于 `-first` 的对。注意跳过重复元素。

## 题目四：接雨水

### 问题描述

给定 `n` 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

### 输入格式

第一行一个整数 `n`，表示柱子的数量。
第二行 `n` 个非负整数 `height[i]`，表示第 i 根柱子的高度。

### 输出格式

一个整数，表示能接的雨水总量。

### 样例输入

```text
12
0 1 0 2 1 0 1 3 2 1 2 1
```

### 样例输出

```text
6
```

### 样例解释

高度图如上，能接 6 个单位雨水。

### 数据范围

- $1 \leq n \leq 2 \times 10^5$
- $0 \leq height[i] \leq 10^5$

### 提示

经典双指针解法：维护左右指针和左右最大高度，按较小的一边计算当前柱子能接的水量。
